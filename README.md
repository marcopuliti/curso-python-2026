Cada clase tiene una parte teórica (1–1,5 hs) con explicación de conceptos y ejemplos, seguida de una parte práctica donde los estudiantes resuelven ejercicios en sus computadoras con acompañamiento docente.

## Evaluación

- **Asistencia:** mínimo 75 %.
- **Evaluación continua:** guías de ejercicios por módulo, con entregas semanales individuales.
- **Proyecto final:** script o pequeña aplicación en Python vinculada al área de investigación del estudiante, usando al menos una herramienta de IA asistida y publicada en GitHub. Presentación oral en la última clase. Ver [docs/proyecto-final.md](docs/proyecto-final.md).

Detalles en [docs/evaluacion.md](docs/evaluacion.md).

## Cómo empezar

1. Seguir la [guía de instalación del entorno](docs/instalacion.md) (Anaconda + VS Code, o Google Colab si preferís no instalar nada).
2. Abrir los notebooks de cada módulo con Jupyter (o subirlos a Colab).
3. Resolver la guía de ejercicios del módulo y entregarla según las pautas de [docs/evaluacion.md](docs/evaluacion.md).
